# 简介

* 模块化设计，数据和展示分离，方便定制UI
* 同时支持AssetsLibrary和PhotoKit,但是AssetsLibrary在框架中已经不再维护
* 增加内存缓存，在内存占用和流畅度上不输常见的图片库
* 支持LivePhoto
* 集成了权限管理，方便定制权限申请的UI
